# hh-system使用说明### 1、clone代码```gitgit clone git@github.com:Zippo-Wang/hh-system.git```### 2、编辑环境变量```bash# 编辑环境变量vim /home/huawei/.bashrc# 填写main.sh所在的路径alias hh="source project_path/main/main.sh"# 使环境变量生效source /home/huawei/.bashrc```### 3、手动修改路径由于windows和Linux文件编码的格式差异与shell引用时使用相对路径的问题，需要用户手动修改`/hh-system/main/main.sh`的`$project_path`### 4、示例```bashhh cms```![img.png](img.png)
